\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Iterators$>$\newline
class iterators\+::impl\+::\+Zip\+Iterator$<$ Iterators $>$}
Class combining multiple iterators into one. Use it to iterate over multiple ranges at the same time. 

Zip\+Iterators only support the operators of the least powerful underling iterator. Zipping a random access iterator (e.\+g. from std\+::vector) and a bidirectional iterator (e.\+g. from std\+::list) results in a bidirectional iterator. All operators are SFINAE friendly.

Zip\+Iterators return a tuple of references to the range elements. When using structured bindings, no additional reference binding is necessary.

Let {\ttfamily z} be a \doxylink{classiterators_1_1impl_1_1ZipIterator}{Zip\+Iterator} composed from two {\ttfamily std\+::vector\texorpdfstring{$<$}{<}int\texorpdfstring{$>$}{>}}
