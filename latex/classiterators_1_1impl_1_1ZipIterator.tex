\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Iterators$>$\newline
class iterators\+::impl\+::\+Zip\+Iterator$<$ Iterators $>$}

Class combining multiple iterators into one. Use it to iterate over multiple ranges at the same time. 

Zip\+Iterators only support the operators of the least powerful underling iterator. Zipping a random access iterator (e.\+g. from std\+::vector) and a bidirectional iterator (e.\+g. from std\+::list) results in a bidirectional iterator. All operators are S\+F\+I\+N\+AE friendly.

Zip\+Iterators return a tuple of references to the range elements. When using structured bindings, no additional reference binding is necessary.

Let {\ttfamily z} be a \mbox{\hyperlink{classiterators_1_1impl_1_1ZipIterator}{Zip\+Iterator}} composed from two {\ttfamily std\+::vector$<$int$>$} 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{auto} [val1, val2] = *z; \textcolor{comment}{// val1 and val2 are references to the vector elements}}
\DoxyCodeLine{val1 = 17; \textcolor{comment}{// this will change the respective value in the first vector}}
\end{DoxyCode}
