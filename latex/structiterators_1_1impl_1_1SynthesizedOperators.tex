\subsubsection*{template$<$typename Impl$>$\newline
struct iterators\+::impl\+::\+Synthesized\+Operators$<$ Impl $>$}
CRTP-\/class that provides additional pointer arithmetic operators synthesized from basic operators. 

Adds the following operators
\begin{DoxyItemize}
\item postfix increment and decrement (requires the respective prefix operators)
\item array subscript operator\mbox{[}\mbox{]} (requires operator+ and dereference operator)
\item binary arithmetic operators (requires compound assignment operators)
\item inequality comparison (requires operator==)
\item less than or equal comparison (requires operator\texorpdfstring{$>$}{>})
\item greater than or equal comparison (requires operator\texorpdfstring{$<$}{<})
